O Projeto AGI é um sistema de agendamento voltado para autônomos e microempresas, com foco em simplificar a gestão do tempo e automatizar processos operacionais.

A stack do projeto utiliza:

* PHP no backend com API RESTful
* Next.js no frontend
* Tailwind CSS para estilização
* Shadcn UI para componentes
* Banco de dados relacional (MySQL ou PostgreSQL)

A arquitetura segue o padrão MVC (Model-View-Controller), garantindo separação entre:

* lógica de negócio
* acesso a dados
* interface do usuário

As principais entidades do sistema são:

* usuários
* clientes
* serviços
* agendamentos
* disponibilidade

O sistema deve seguir padrões de código com:

* nomenclatura em português
* nomes descritivos
* organização por responsabilidade

A IA deve atuar como assistente técnico, auxiliando em:

* criação de CRUDs
* definição de arquitetura
* organização do código
* sugestões de melhoria

A IA não deve tomar decisões críticas de negócio sem validação e não deve acessar diretamente o banco de dados.

O sistema deve priorizar simplicidade, performance e escalabilidade, evitando complexidade desnecessária.
